# 📸 Guide til at tilføje billeder

## Trin 1: Download billeder fra deres hjemmeside
Besøg https://snedkerfirmaet-lysen.dk/ og højreklik på de billeder du vil bruge:

### Foreslåede billeder til projekter:
- **Terrasse i Vig** → gem som `images/terrasse-vig.jpg`
- **Vinduer i Ishøj** → gem som `images/vinduer-ishoej.jpg`
- **Køkken renovering** → gem som `images/koekken-renovering.jpg`
- **Plankeværk** → gem som `images/plankevaerk.jpg`
- **Rumdeler** → gem som `images/rumdeler.jpg`
- **Badeværelse** → gem som `images/bad-renovering.jpg`
- **Parkbænke** → gem som `images/parkbaenke.jpg`
- **Facade** → gem som `images/facade-renovering.jpg`

### Andre billeder:
- **Martin Lysén** → gem som `images/martin-lysen.jpg` (til om-os siden)
- **Logo** → gem som `images/logo.png`
- **Værksted/kontor** → gem som `images/vaerksted.jpg`

## Trin 2: Placér billeder
Alle billeder skal placeres i `images/` mappen.

## Trin 3: Automatisk fallback
Websitet er sat op til automatisk at vise placeholder ikoner hvis billeder mangler, så det virker også uden billeder.

## Optimal billedstørrelse
- **Bredde:** 800-1200 pixels
- **Format:** JPG eller WebP
- **Kvalitet:** Medium til høj (for hurtigere loading)

## Billeder allerede understøttet i koden:
✅ Projektgalleri billeder med fallback
✅ Responsive design
✅ Hover effekter
✅ Alt-tekst for tilgængelighed
